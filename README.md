## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Description

Setting up user authentication with route protection using jwt (set up the database before launching the application)

## Technology
- Nestjs
- postgresql
- Docker
- TypeOrm
- Swagger
